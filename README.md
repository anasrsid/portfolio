# Anas Rahman Siddiqui — Portfolio

Personal portfolio website for **Anas Rahman Siddiqui**, a videographer and video editor based in Karachi, Pakistan.

## Structure

```
├── index.html      — Homepage (hero, about, portfolio, services, channels, contact)
├── works.html      — Full portfolio (horizontal videos + vertical shorts)
├── style.css       — All styles
├── script.js       — Animations, nav, interactions
├── images/         — Hero image and assets
└── .gitignore
```

## Tech

- Static HTML/CSS/JS — no build tools, no frameworks
- Inter font via Google Fonts
- YouTube embeds for portfolio videos
- WhatsApp contact button
- Responsive (mobile-first)

## Local development

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.
